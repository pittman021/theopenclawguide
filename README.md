## 🚀 Quick Start

### Option 1: Interactive Course (Recommended)

1. **Open this folder in OpenClaw**
2. **Tell your agent:**
   > "Read START-HERE.md and start lesson 1"
3. **Learn by doing** — Your agent guides you through each lesson

### Option 2: Read the Guide

- **Web version:** Open `index.html` in your browser
- **PDF version:** Run `./generate-pdf.sh` to create a PDF

## 🎯 Who Is This For?

This guide is designed for **CS Directors, Revenue Ops, and Sales Leaders** who want AI automation without engineering teams.

You're a good fit if:
- You're drowning in manual follow-ups
- You miss renewal windows because of volume
- You want AI leverage but don't code
- You learn by doing, not just reading
